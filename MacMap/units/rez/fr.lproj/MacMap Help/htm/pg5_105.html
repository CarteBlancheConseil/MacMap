Geog Englobant

Recherche les objets du type surfacique spécifié englobant les objets contrastées (ou à défaut pré-sélectionnées).
L'option "en partie" permet de retenir tous les objets englobant au moins un point des objets sources.
L'option "en totalité" permet de ne retenir que les objets englobant tous les points des objets sources.
L'option "strictement" permet de ne pas retenir les objets dont certains points des objets sources se trouvent sur la limite.

Commande script 
<include>
    <bool>En totalité (0/1)</bool>
    <bool>Strictement (0/1)</bool>
</include>
