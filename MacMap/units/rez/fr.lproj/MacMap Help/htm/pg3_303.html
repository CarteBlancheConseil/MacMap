XMap Cohérence topologique

La cohérence topoloqique permet de conserver la relation géographique entre é objets. Si une ligne aboutit à un point, le déplacement de ce point conduira à une modification de la ligne pour quelle aboutisse toujours sur ce point. Cette procédure est automatique.
Le case à cocher "actif" permet d'activer la détection de modification.
Le pop-up "Sur les types" permet de définir les types concernés par la cohérence topologique.
