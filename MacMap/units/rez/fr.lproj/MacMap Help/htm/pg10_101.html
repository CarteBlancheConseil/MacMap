XMap Intersection

Permet de calculer les intersections entre les objets sélectionné, 2 à 2. Ces objets peuvent êtres de genre linéaire ou surfacique.
Les intersections résultantes sont créées dans le type courant (boîte présentation), qui peut être de genre surfacique, linéaire ou ponctuel.

Commande script 
<intersection>
    <name>Nom du type</name>
</intersection>
