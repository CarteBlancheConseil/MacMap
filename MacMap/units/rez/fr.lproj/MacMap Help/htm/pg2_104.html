Tool Visu-bulle
Utilisation

Cet outil permet d'afficher les attributs des objets dans une fenêtre flottante lorsq'on les survoles avec le curseur.
Un clic permet de sélectionner l'objet survolé.
Un double-clic permet d'ouvrir la fiche de l'objet survolé.

Pop-up

Cet outil ne gère pas de pop-up.

Options

Les options de cet outil permettent de définir pour chaque type la liste des champs à afficher.
