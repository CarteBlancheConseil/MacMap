XMap Gestion des types

Le gestionnaire de type permet toutes les manipulations sur les types d'une base de données. La liste présente les types présents dans la base de données.
Les boutons permettent les opérations suivante : - Nouveau : création d'un type. C'est un raccourci vers la fonction "Créer un type".
- Ouvrir : ouverture d'un type existant. C'est un raccourci vers la fonction "Ouvrir un type".
- Retirer : indique que le type sélectionné doit être retiré à la fermeture de la base de données
- Renommer : indique que le type sélectionné doit être renommé à la fermeture de la base de données
- Protéger : empêche toutes modifications sur le type sélectionné.
- Déprotéger : autorise les modification sur le type sélectionné.
