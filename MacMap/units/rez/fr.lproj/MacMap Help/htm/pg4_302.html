XMap Créer un polygone

Cette fonction permet de créer un polygone dans le type sélectionné dans la boîte de présentation à partir d'un centroïde, d'un rayon et d'un nombre de cotés. Si un objet est constrasté, le centroïde est initialisé à la coordonnée du premier point de cet objet, sinon, il est initialisé à la coordonnée du centre de l'écran.

Commande script 
<makepoly>
    <name>Nom du type</name>
    <float>X</float>
    <float>Y</float>
    <float>Taille</float>
    <int>Nombre de cotés</int>
</makepoly>
