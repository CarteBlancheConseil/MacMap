Tool Sélection
Utilisation

Cet outil permet de sélectionner les objets du document à l'aide de la souris, par simple clic ou par zone (rectangle, cercle ou polygone). Pour les sélections par polygone, la contrainte d'angle est obtenue avec la touche "shift".
Si à la fin d'une sélection, la touche "shift" est maintenue, la sélection est ajoutée à la sélection existante. Les objets appartenant à la sélection initiale et à la nouvelle sélection sont désélectionnés.
Si la touche "shift" n'est pas maintenue, la sélection est remplacée.
Le double-clic permet d'ouvrir la fiche sur la sélection.
Les objets représentés par des styles non sélectionnables ne peuvent pas êtres sélectionnés (cf. Boîte de présentation).

Pop-up

Le pop-up permet de choisir le type de sélection par zone :
Sélection par rectangle : cliquer-déplacer.
Sélection par cercle : cliquer-déplacer.
Sélection par polygone : succession de clics, puis "Entrée" ou double clic pour valider la sélection.
Si l'on choisit la sélection par polygone, la sélection par pointé est désactivée. Pour réactiver la sélection par pointé, il faut choisir un des deux autres modes.

Options

4 options sont disponibles : Sélectionner tous les objets : si plusieurs objets du même type sont superposés, seul le premier objet est sélectionné si cette option est désactivée. Dans le cas contraire, tous les objets sont sélectionnés.
Sélectionner tous les type : si plusieurs objets de types différents sont superposés, seuls les objets du premier type (le dernier affiché dans la boîte de présentation) sont sélectionnés si cette option est désactivée. Dans le cas contraire, tous les objets sont sélectionnés.
Objets inclus entièrement : lors d'une sélection par zone, ne prend en compte que les objets dont tous les points de la segmentation sont inclus dans la zone de sélection.
Utiliser la contrainte d'angle : lors d'une sélection par polygone, permet d'activer la contrainte d'angle, pour la valeur désirée (exprimée en degrés).

Les deux listes présentes aux rubriques "Sélection par pointé" et "Sélection par zone" permettent de filtrer les types sélectionnables dans ces deux modes. Ces filtres s'ajoutent au filtrage préexistant de la boîte de présentation.
