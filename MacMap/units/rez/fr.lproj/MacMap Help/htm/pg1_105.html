XMap Import Raster

Sélectionnez l'image à importer (TIF, JPEG, PNG, ou GIF).
Si cette image dispose de paramètres de géoréférencement valides (formats TAB ou TFW), l'import se fait automatiquement.
Si cette image ne dispose pas de paramètres de géoréférencement, la fonction vous demandera de définir la coordonnée du coin supérieur gauche de l'image, ainsi que définir la coordonnée du coin inférieur droit ou l'échelle de l'image.
Les objets sont importés sous forme de rectangles correspondant à l'emprise de l'image, dans le type Raster défini dans la grille d'import. Si l'on redimensionne le rectangle support, l'image est redimensionnée en conséquence. Le nom de l'image est défini dans le champ Nom de l'objet.
Les images sont dupliquées dans la table correspondante (dossier Contents/Resources/rasters/).
Pour rester cohérent géographiquement, une image importée doit être dans la même projection que la base de donnée avec laquelle vous travaillez.
