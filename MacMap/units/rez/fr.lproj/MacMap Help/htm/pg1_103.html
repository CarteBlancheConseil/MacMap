XMap Grille d'import

La grille d'import permet de spécifier le type utilisé pour recueillir les objets créés lors d'un import. Quatre types peuvent être utilisés pour chaque genre d'objet': Linéaires, Surfaciques, Ponctuels, Textuels, Rasters.
Le bouton "Projection" vous permet d'accéder à une fenêtre permettant de définir la projection des données en entrée. Ceci permet, si la projection du document a été réglé de re-projeter dynamiquement les données à l'import.
Commande script 
<igrid>
    <int>Identifiant EPSG de projection</int>
    <name>Nom du type ponctuel</name>
    <name>Nom du type linéaire</name>
    <name>Nom du type textuel</name>
    <name>Nom du type surfacique</name>
    <name>Nom du type raster</name>
</igrid>
