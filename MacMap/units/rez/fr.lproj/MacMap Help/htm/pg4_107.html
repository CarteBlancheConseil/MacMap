XMap Associer

Cette fonction permet d'associer chacun des objets sélectionnés (ligne ou surface), si la distance les séparants est inférieure à la distance spécifiée.
Les objets associés en fin de traitement auront les attributs du premier objet sélectionné. Les objets non associés sont contrastés.
L'option "Effacer les objets originaux" supprime les objets sélectionés.

Commande script 
<association>
    <float>Distance</float>
    <bool>1 pour détruire les objets après association, 0 sinon</bool>
</association>
