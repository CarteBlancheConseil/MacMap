XMap Contraintes de saisie

Permet de définir des contraintes de saisie sur un champ. Les contraintes peuvent être de trois types :
- Naturelle : le type de la contrainte correspond au type du champ, par exemple on définie des valeurs entières sur un champ de type "Entier".
- Champ classe : les contraintes correspondent à des intitulés. Dans ce cas, la valeur du champ sera correspondra à l'index de la contrainte, comme c'est le cas pour les champs sous-types et couleur.
- Valeurs multiples : uniquement sur les champs "chaîne de caractère" et "entier". Dans le cas d'un entier, on peut définir jusqu'a 32 valeurs, et le champ est codé bit par bit. Dans le cas d'une chaîne, on obtient une suite de 0 et de 1.
Il n'est pas possible de supprimer ou de modifier une contrainte sur une table contenant des objets.
