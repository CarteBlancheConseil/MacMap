XMap Export ShapeFile

Permet d'exporter les objets au format ShapeFile (format ArcView™).
Si il existe une sélection, cette sélection est exportée en autant de tables que cette sélection contient de types. Si il n'existe pas de sélection, tous les objets du projet sont exportés en autant de tables qu'il existe de types. Les tables ShapeFile générées portent le nom des types.
Si les informations de projections (cf. Format d'export et Préférences) ont été réglés, les données sont re-projetées dynamiquement à l'export.

Commande script 
<shpexport>
    <path>Chemin d'accès au dossier (POSIX)</path>
</shpexport>
