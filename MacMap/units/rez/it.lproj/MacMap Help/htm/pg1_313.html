XMap Collegamento dati

Permette di realizzare un collegamento su dei dati esterni contenuti in un file formato CSV, Testo tabulato o DBF. I campi del file esterno sono aggiunti nella scheda di MacMap, al di sotto dei campi che contengono gli attributi. Questi campi non sono modificabili, possono però essere eliminati dalla lista dei campi del formato scheda (senza modificare il file estermo).
Per realizzare un collegamento, è sufficiente definire un campo chiave tra il tipo MacMap interessato e il file esterno.
