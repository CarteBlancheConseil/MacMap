vDef Direzione

La vDef Direzione permette di rappresentare gli oggetti lineari o superficiali con una freccia. Quando il campo "Direzione" dell'oggetto è 1, la freccia sarà situata all'estremità della segmentazione. Per un valore uguale a -1, la freccia sarà situata all'inizio della segmentazione, in senso inverso. Se il campo direzione è uguale a 0, la freccia non è rappresentata. Per default, lo stile è basato sul campo "Sotto-tipo".

Grafica

I parametri della rubrica "Grafica" si applicano individualmente, per ciascun Sotto-tipo visualizzato.
Il popup "Direzione" permette di specificare il campo contenente l'orientamento (-1,0 a 1,0).
Il campo "Dimensione" permette di specificare la dimensione della freccia.
Il campo "Interno" permette di scegliere un colore o un retino per l'interno.
Il campo "Opacità" permette di regolare la trasparenza dell'interno.
Il campo "Spessore" permette di definire le dimensioni del bordo.
Il campo "Colore" permette di specificare il colore del bordo.
Il campo "Spostamento" permette di fissare uno spostamento della freccia (a destra per un valore positivo, a sinistra per un valore negativo).

Vista

I parametri della rubrica "Vista" si applicano individualmente, per ciascun Sotto-tipo visualizzato.
I due pop-up permettono di regolare la sequenza delle scale alle quali lo stile sarà visibile.

Opzioni : Parametri generali

Il campo "Campo classe" permette di specificare l'informazione che lo stile deve rappresentare. Si possono utilizzare solo dei campi numerici che dispongono di valori predefiniti.
Se si cambia il campo classe di uno stile già esistente, i parametri di visualizzazione vengono reinizializzati.
Il campo "Scala di riferimento" permette di ottenere una vista proporzionale. Tutte le dimensioni saranno ricalcolate in funzione della scala in corso e della scala di riferimento. Lasciando questo campo al valore "Dimensione fissa" si ottiene la stessa rappresentazione per tutte le scale.
Il campo "Unità" permette di scegliere tra una espressione dei valori (dimensione, spessore,...) in pixel o in millimetri.
Il campo "Inverti l'ordine" permette di invertire l'ordine di visualizzazione delle classi.
Il campo "CMYK" permette di scegliere il tipo di colore utilizzato : CMYK o RGB.

Opzioni : Mostra l'oggetto se

Questa opzione permette di fissare le condizioni di visualizzazione, per ciascun oggetto. Una condizione di visualizzazione si compone di tre elementi :
- Un campo del formato scheda,
- Un operatore,
- Un valore.
Se l'oggetto non risponde alle condizioni definite non sarà visualizzato dallo stile, senza necessità di essere mascherato.

Opzioni : Offset

Questa opzione permette di scegliere due campi x e y che modificano il punto di visualizzazione senza però modificare la geometria dell'oggetto. Questi campi possono essere compilati utilizzando lo strumento Offset.

Applica

Registra le modifiche allo stile.
Se lo stile è rappresentato nella vista in corso, le modifiche sono subito applicate, senza la necessità di uscire dall'editor.
Per uscire dall'editor senza salvare le modifiche è sufficiente chiuderlo o premere "esc" sulla tastiera.
