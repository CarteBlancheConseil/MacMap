XMap - Toolbox

The toolbox lets you access various MacMap tools.
Clicking on a tool button will activate it.
Click and hold on a tool button to access the corresponding menu, if there is one.
You can access the options for a tool (if it has any) by double-clicking on the tool button.
When the window is on top, you can move to the next tool in the toolbox by pressing the tab key.
The first button in this box is a shortcut to the tool currently being used.
The second button allows access to the XMap configuration menu tools.
The third button is used to activate cursor tracking.
The box is divided into three sections which you can access by clicking on the appropriate button.
