XMap - Translate

This function lets you translate the selected objects in x and in y coordinates. Two modes are available:
Translate by Offset: objects are moved by x and y units.
Translation to Coordinate: objects are moved so that their first point matches the indicated coordinate. If an object is contrasted, the x and y fields are pre-filled with the coordinate of the first point of the segmentation of this object.

Script Command 
<translation>
    <float>Translation in X</float>
    <float>Translation in Y</float>
    <int>1(=translation by offset)|2(=translation to a coordinate)</int>
</translation>
