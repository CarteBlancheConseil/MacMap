XMap - Associate

This function lets you associate each selected object (line or area) if the distance between them is less than the specified distance.
The objects associated after processing will have the attributes of the first selected object. The unassociated objects are contrasted.
The "Erase Original Objects" option deletes the selected objects.

Script Command 
<association>
    <float>Distance</float>
    <bool>1 to destroy objects after association, otherwise 0</bool>
</association>
