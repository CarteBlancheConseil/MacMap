XMap - Publish as TIFF

This function lets you export the print area in TIFF format. The print area is displayed on the map with a black, dashed border. This area can be modified with the Print Area tool.
Options
The options let you tile the published area by specifying the number of images you wish to display horizontally and vertically. This function exports then a series of numbered contiguous images.
You can also specify the compression quality and the image resolution.
The "Georeferenced" option allows you to export the georeferencing information from the image in a second file in TFW format.
Script Command 
<tiffpublisher>
    <path>Folder path (POSIX)</path>
    <name>File name</name>
    <int>Number of horizontal cells (1 to n)</int>
    <int>Number of vertical cells (1 to n)</int>
    <int>Compression quality (1=min to 6=max)</int>
    <int>Resolution in ppp</int>
    <int>1 to export the georeferencing information, otherwise 0</int>
</tiffpublisher>
