XMap - Dash Manager

The dash manager lets you import XML files describing dashes into a type. These dashes can then be used in vDefs.
The "+" button lets you open an XML file saved on disk.
The list lets you select a dash. Its representation is displayed in the lower portion of the window.
By clicking Select, you return the selected dash to the calling vDef.

List of vDefs which use dashes:
Line View
Proportional Lines
Colored Lines
