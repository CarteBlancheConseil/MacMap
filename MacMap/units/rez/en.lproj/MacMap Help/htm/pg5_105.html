Geog - Encompassing

Search for objects of the specified area type encompassing contrasted objects (or pre-selected if none exist).
The "Partially" option lets you retain all objects encompassing at least one point from the source objects.
The "Entirely" option lets you retain only those objects encompassing all points of the source objects.
The "Strictly" option lets you not retain objects for which some points of the source objects are located on the boundary.

Script Command 
<include>
    <bool>Entirely (0/1)</bool>
    <bool>Strictly (0/1)</bool>
</include>
