XMap - Transfer of Type

The options available let you erase the selection after transfer, and create a similar structure of fields in the receiver type.
This function operates on a selection of a single type.
Script Command 
<transtype>
    <name>Type name</name>
    <bool>1 to preserve the original objects, 0 to delete them</bool>
    <bool>1 to create fields, otherwise 0</bool>
</transtype>
