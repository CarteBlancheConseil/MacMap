Geog - Inside

Search for objects of the specified type located in at least one of the contrasted areas (or pre-selected if none exist).
The "Partially" option lets you retain all objects for which at least one point is located in one of the areas.
The "Entirely" option lets you retain only those objects for which all points are located in one of the areas.
The "Strictly" option lets you not retain objects for which some points are located on an area boundary.

Script Command 
<inside>
    <bool>Entirely (0/1)</bool>
    <bool>Strictly (0/1)</bool>
</inside>
