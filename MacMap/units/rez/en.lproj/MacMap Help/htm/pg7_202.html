XMap - Pattern Manager

The pattern manager lets you import PDF files into a type. These patterns can then be used in the vDefs to generate raster images.
The "+" button lets you open a PDF file saved on disk.
The list lets you select a pattern. Its representation is displayed in the lower portion of the window.
By clicking Select, you return the selected pattern to the calling vDef.

List of vDefs which use patterns:
