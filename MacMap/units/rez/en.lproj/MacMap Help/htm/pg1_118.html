XMap - Mif-Mid Export

Allows you to export objects in Mif-Mid format (MapInfoâ„¢ export format).
If a selection exists, this selection is exported in as many tables as it has types. If there is no selection, all of the objects in the project are exported in as many tables as there are types. The Mif-Mid tables generated have the same names as the types.
If the projection information (see Export format and Preferences) has been set, the data is dynamically reprojected on export.
