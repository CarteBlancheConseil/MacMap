MacMap Text Export

This function lets you export objects in MacMap Text (.MMT) format.
If a selection exists, this selection is exported in as many tables as there are types. If there is no selection, all of the objects in the project are exported in as many tables as there are types. The generated tables have the same names as the types.
If the projection information (see Export format and Preferences) has been set, the data is dynamically reprojected on export.
