XMap - Create Rectangle

This function lets you create a rectangular area in the type selected in the presentation box starting from a centroid and its dimensions. If an object is contrasted, the centroid is initialized to the coordinate of the first point of that object; otherwise, it is initialized to the coordinate at the center of the screen.

Script Command 
<makerect>
    <name>Type name</name>
    <float>X</float>
    <float>Y</float>
    <float>Width</float>
    <float>Height</float>
</makerect>
