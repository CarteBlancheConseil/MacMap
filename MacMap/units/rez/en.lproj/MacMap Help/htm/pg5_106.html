Geog - Along

Search for objects of the specified line type located along the contrasted objects (or pre-selected if none exist).
The "Partially" option lets you retain all objects for which at least two consecutive points are located along the source objects.
The "Entirely" option lets you retain only those objects for which all points are located along the source objects.
The "Strictly" option lets you not retain objects for which some points are not precisely at the boundary of one of the source objects.

Script Command 
<along>
    <bool>Entirely (0/1)</bool>
    <bool>Strictly (0/1)</bool>
</along>
