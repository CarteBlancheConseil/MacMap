Offset Tool
Use

The Offset tool lets you move the object representation if the style uses a double "offset" field (x and y).
A click and drag lets you then relocate the object representation. The offset x and offset y fields for the object are filled in with the appropriate values. The object segmentation is not modified.

Popup

This tool does not have any popups.

Options

This tool has no options.
