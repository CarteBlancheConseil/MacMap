XMap - String Processing

Allows you to carry out logic operations on strings. It operates on a single-type selection.
The first list in the window shows the list of fields of the type you are working with. The set of buttons makes it possible to select one or more fields that will be used as a source. This means you can concatenate several fields and, if necessary, separate values with a separator.
The second part of the window allows you to carry out various operations on the string resulting from concatenating the different sources:
- Change case,
- Find-Replace,
- Formatting (for example, the value "14" formatted using "T####" will give "T0014").
The last pop-up allows you to specify the field that will be used to store the result of the processing.
Script Command 
<stringprocessing>
    <container>
    <!--List of fields to concatenate-->
        <name>Field name</name>
    <!--This tag must be repeated for each field to concatenate-->
    </container>
    <string>Separator</string>
    <int>Change case (1=No change, 2=Lower case, 3=Upper case, 4=Capitalize, 5=Capitalize words)</int>
    <string>String to be found</string>
    <string>Replacement string</string>
    <string>Format (e.g. 12 in AZE#####RTY format will give AZE00012RTY. If the value exceeds the space, it is truncated to respect the length of the format).</string>
    <name>Name of field to be filled</name>
</stringprocessing>
