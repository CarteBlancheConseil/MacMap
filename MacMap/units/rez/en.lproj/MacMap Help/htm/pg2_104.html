Tooltip Tool
Use

This tool lets you display attributes for objects in a floating window when you move over them with the cursor.
Click to select the object beneath the cursor.
Double-click to open the card for the object beneath the cursor.

Popup

This tool does not have any popups.

Options

The options for this tool let you define, for each type, the list of fields to be displayed.
