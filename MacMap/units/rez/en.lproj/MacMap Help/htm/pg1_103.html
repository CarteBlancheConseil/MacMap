XMap - Import Grid

The import grid lets you specify the type used to collect the objects created during import. Four types can be used for each kind of object: Line, Area, Point, Text, and Raster.
The "Projection" button allows you to access a window where you can set the incoming data projection. If the document projection has been set, this allows you to dynamically reproject the data on import.
