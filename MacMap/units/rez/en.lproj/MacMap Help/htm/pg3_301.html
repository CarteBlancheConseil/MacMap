XMap - Cleanup Options

The cleanup options let you define the settings for the Clean function. The "Clean" function lets you make automatic corrections to a network in order to make it topological. These settings are divided into 5 sections:

Types involved in cleanup:
"Nodes" are points marking each intersection, while "edges" are lines composed of a point at each end.

Intersection Creation:
Crossroads are intersections or ends of the network. Pre-existing crossroads are taken into account. If some intersections do not include crossroads, they will be created in the type and subtype and with the defined name.

Intervention Distances:
Intervention distances to crossroads let you specify the limit from which two points belong to the same intersection. Intervention distances to bridges let you specify the limit from which two points belonging to the same intersection should not be considered as a crossroad. This option lets you, for example, avoid automatic splitting of crossing segments if their intersection is found in the intervention radius of a bridge.

Bridge Definition:
This section lets you specify in which type and subtype the cleanup must search for bridges. The bridges must be pre-existing.

Automatic Cleanup:
Enabling automatic cleanup lets you run the cleanup procedure each time you create or edit an object belonging to the network (node or edge). This function must be used with care since any edits are automatic. It also uses a lot of processor time.
