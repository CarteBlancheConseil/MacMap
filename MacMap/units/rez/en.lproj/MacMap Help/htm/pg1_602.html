XMap - Spreadsheet

The spreadsheet lets you view and modify the attribute table for one or more selected objects of the same type. Each field can be individually modified.

By default, fields are editable in text mode. In this mode, the "Enter" key lets you validate the modification in progress; the "Esc" key is used to end the edit without saving it.
Fields with entry constraints are editable via a popup.
Boolean fields are editable by clicking in the cell.
Hidden fields are not visible.
Protected fields are not editable.
You can also display Calc results in the spreadsheet. That result is not editable.

The "Esc" key lets you close the spreadsheet if no editing is in process.

Click on a row or column heading to select it. You can use the "Shift" and "Apple" shortcut keys.
Click on a column heading to change the displayed attribute for this column via a popup.
Click on a column separator to resize the column.
Double-clicking on a row heading lets you open the object's card.
Click and hold on the upper left cell to access the spreadsheet functions.

Spreadsheet Functions:
Insert Column: Insert an empty column before the first selected column.
Remove Column: Remove the selected columns from the spreadsheet.
Select All Columns: Select all columns in the spreadsheet.
Insert Objects: Add to the spreadsheet the selected objects which are not already in it.
Remove Objects: Remove the selected objects from the spreadsheet.
Select All Objects: Select all objects in the spreadsheet.
Sort: Sort the selected objects (or all objects in the spreadsheet if there is no selection) in the selected fields (in the order in which they were selected). A dialog box lets you specify the sort characteristics (Numeric/Alphanumeric Ascending/Descending). Holding down the "Shift" key when you activate this function will prevent the dialog box from opening: The objects are then sorted in ascending order, and based on the field type.
Export: Export the values of the selected fields for the selected objects (or for all objects in the spreadsheet if there is no selection). You can export in DBF, tab delimited, CSV or SQL format.
