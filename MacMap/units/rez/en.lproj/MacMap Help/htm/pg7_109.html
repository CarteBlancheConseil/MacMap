vDef - Proportional Lines

The Proportional Lines vDef lets you represent objects as lines whose width is proportional to the value of a field. Objects are displayed in order of decreasing value.
Statistics

The "Field" section lets you specify the field on which the view will be based. This field must contain numeric values.
The "Proportionality" section lets you specify the proportionality type that you wish to obtain. With W=Q, you calculate the width of lines relative to the quantity. For W=Q3/2, you calculate the width of lines relative to the quantity raised to the power of 3/2. It is the same for other values of W.
The "Min calculation" popup lets you calculate automatically the lower limit for the representation of objects (based on type, selection or contrasts). This value corresponds to the smallest value of the field that you wish to represent.
The "Max calculation" popup lets you calculate automatically the upper limit for the representation of objects (based on type, selection or contrasts). This value corresponds to the largest value of the field that you wish to represent.

Appearance

The "Minimum Thickness" field lets you set the line width corresponding to the specified minimum value. In the "Maximum Thickness" field, you obtain the width of lines corresponding to the specified maximum value.
The "Offset" field lets you offset the line (to the right for a positive value and to the left for a negative value).
The "Miter Limit" field lets you set the limit at which joins are beveled.
The "Smoothing" field lets you smooth the segmentation (remove angularities).
The "Interior" field lets you select a color or a gradient. Selecting a gradient lets you color lines based on another field. Clicking on the color enables the Gradient panel, which lets you define a substyle.
The "Cap" field lets you select the type of line ending (butt, round or projecting).
The "Join" field lets you select the representation type for each of the joins of the segmentation (miter, rounded or bevel).
The "Dash" field lets you select a dash style.

Visibility

The two popups let you set the scale range for which the style is visible.

Gradient

The Gradient panel lets you create substyles. The definition of these substyles consists of discretizing a statistical series and then assigning a color to each of the classes.

The "Field" section lets you specify the field on which the view will be based. This field must contain numeric values.
The "Discretization" section lets you select an automatic method for defining classes.
The "Number of Classes" section lets you specify the number of classes desired.
The "Min calculation" popup lets you calculate automatically the lower limit for the representation of objects (based on type, selection or contrasts). This value corresponds to the smallest value of the field that you wish to represent.
The "Max calculation" popup lets you calculate automatically the upper limit for the representation of objects (based on type, selection or contrasts). This value corresponds to the largest value of the field that you wish to represent.
The "Discretize" button lets you calculate the limits for classes, based on the specified settings. This calculation can be done based on type, selection or contrasts. The result is displayed in the list and can be modified manually if necessary (by clicking on a value in the list).
You can add or remove classes by using the "+" and "-" buttons.
The "Colorimetry" section lets you select the colors to apply to each of the classes. These colors are automatically calculated based on the reference colors.
The associated wells let you edit the color of the class or the pattern.

Options: General

The "Scale of Reference" field lets you set the proportionality based on the scale. Each of the sizes indicated is recalculated when displayed according to the current scale and reference scale.
The "Unit" field lets you select between various measurements (size, thickness, ...) in pixels or in millimeters.
The "CMYK" field lets you set the type of color you want: CMYK or RGB.

Options: Display object if

This option lets you define display conditions for each object. A display condition is made up of three elements:
- A field in the attribute table,
- An operator,
- A value.
An object not meeting the defined condition will not be displayed by the style, without having to hide it.

Options: Offset

This option lets you select two x and y fields which modify the location of the display without modifying the object geometry. These fields can be filled in using the Offset tool.

Apply

This button saves the changes to the style and apply it to the current view if this style is shown at screen.
