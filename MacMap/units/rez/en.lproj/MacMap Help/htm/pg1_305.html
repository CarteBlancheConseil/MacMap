XMap - Table Structure

This function lets you manage the structure of MacMap tables. This function lets you:
- Add a new field to a table (integer, floating, boolean, date, time, string).
- Delete an existing field.
- Edit an existing field (change the name, type, length).
- Specify a default value.
- Set the protection level (hidden, uneditable, undeletable field).
