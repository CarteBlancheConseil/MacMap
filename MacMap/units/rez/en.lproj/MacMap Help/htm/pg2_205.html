Simple Shapes Tool
Use

This tool lets you enter new linear or area objects in the type selected in the Presentation Box. The tool can be used either by clicking and dragging or by double-clicking. In the first case, the size is defined by moving the mouse while, in the second case, it is set from a dialog box.

Popup

The popup lets you choose between a square shape and a round shape.

Options

The options let you:
- choose between a shape centered on the click or an origin at the click,
- automatically open the card every time an object is created,
- indicate the measurements in cm to the current scale.
