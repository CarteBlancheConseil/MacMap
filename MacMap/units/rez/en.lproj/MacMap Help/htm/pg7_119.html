vDef - Proportional symbols

The Proportional symbols vDef lets you represent objects as icons proportional to the value of a field, located at the centroid of objects (if no coordinate is specified in the vDef options). The proportionality is calculated based on the area. Objects are displayed in order of decreasing value.

Statistics

The "Field" section lets you specify the field on which the view will be based. This field must contain numeric values.
The "Proportionality" section lets you specify the proportionality type that you wish to obtain. With A=Q, you calculate the area or surface of symbols relative to quantity. For A=Q2/3, you calculate the area of symbols relative to the quantity raised to the power of 2/3. It is the same for other values of S.
The "Min calculation" popup lets you calculate automatically the lower limit for the representation of objects (based on type, selection or contrasts). This value corresponds to the smallest value of the field that you wish to represent.
The "Max calculation" popup lets you calculate automatically the upper limit for the representation of objects (based on type, selection or contrasts). This value corresponds to the largest value of the field that you wish to represent.

Appearance

The "Minimum height" field lets you set the size of the symbol corresponding to the specified minimum value. In the "Maximum height" field, you obtain the height of the symbol corresponding to the specified maximum value.
The "Symbol" button lets you select a symbol using the Icon Manager. The original size of the symbol is indicated below this button.

Visibility

The two popups let you set the scale range for which the style is visible.

Options: General

The "Scale of Reference" field lets you set the proportionality based on the scale. Each of the sizes indicated is recalculated when displayed according to the current scale and reference scale. The "Unit" field lets you select between various measurements (size, thickness, ...) in pixels or in millimeters.

Options: Display object if

This option lets you define display conditions for each object. A display condition is made up of three elements: - A field in the attribute table, - An operator, - A value. An object not meeting the defined condition will not be displayed by the style, without having to hide it.
Options: Offset

This option lets you select two x and y fields which modify the location of the display without modifying the object geometry. These fields can be filled in using the Offset tool.

Apply

This button saves the changes to the style and apply it to the current view if this style is shown at screen.
