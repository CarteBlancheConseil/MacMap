XMap - Fill

This function lets you assign values to a list of fields for each object contained in the selection. You can access this function only if the selection contains objects belonging to a single type. You can fill a field with a value, the result of a function (Calc), or with the contents of another field.
This function operates on a selection of a single type.
Script Command 
<datafill>
    <name>Field name</name>
    <int>1(=value)|2(=calc)|3(=field)|4(=variable value)</int>
    <string>Value|Calc name|Field name|Message</string>
</datafill>
