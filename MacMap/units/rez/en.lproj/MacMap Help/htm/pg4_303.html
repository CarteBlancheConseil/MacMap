XMap - Graticule

This tool lets you create a graticule in the type selected in the Presentation Box. The origin lets you define the upper left corner of the area to be squared off. Next, specify the number of graticule cells, horizontally and vertically, as well as the size and shape of the cells (which can be square or hexagonal).

Script Command 
<makesquaring>
    <name>Type name</name>
    <int>1(=square)|2(=hexagonal)</int>
    <float>X</float>
    <float>Y</float>
    <int>Number of horizontal cells</int>
    <int>Number of vertical cells</int>
    <float>Cell size</float>
</makesquaring>
