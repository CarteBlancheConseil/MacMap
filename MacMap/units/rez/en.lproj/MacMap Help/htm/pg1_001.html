XMap - Preference

Indicate the unit in which distances will be expressed in this document (e.g. meter). This unit may be different than the base unit of the document: For example, you might have a document in centimeters and want to display distances in meters. The resolution of the document and the resolution unit are indicated at the top of the window.
Indicate the number of decimals for displaying distances.
The fields X Origin and Y Origin let you artificially translate the coordinates of objects from the database.
Indicate the document projection. You can directly indicate the EPSG code for the projection, if you know it. This feature is optional but it is necessary for the proper functioning of certain functions (KML Export for example). Once set, the projection of the document is final.
You can also set a background color for the main window. This color will appear when it is printed and published.
Finally, the "Save Object States" option allows you to save object states (selected, contrasted, hidden...) on closing the database.
