Entry Tool
Use

This tool lets you enter new objects in the type selected in the Presentation Box. For line or area types, the entry ends by double-clicking or by pressing the "Enter" key.
This tool uses join options to ensure correct entry.
Popup

This tool does not have any popups.

Options

The options involve joins. You can enable or disable each of 4 types of joins and you can specify the types to be considered. The different joining procedures can be enabled by specifying an activation key in the fields next to the check boxes.
The "Open Card" option allows the automatic opening of the card every time an object is created.
