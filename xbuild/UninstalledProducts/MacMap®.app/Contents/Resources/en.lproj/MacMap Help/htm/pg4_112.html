XMap - Associate by field

This function allows you to aggregate a series of selected linear objects (single type selection) by proximity and based on a field.
The first pop-up allows you to define the field that will serve as a reference for the aggregation (a road name, for example).
The "Sort by Subtype" option allows you to prevent association at intersections if routes of a superior subtype are found (the first subtype being considered superior to the second).
The "Minimum length" field allows you to limit the minimum length of a section, if possible.
The "Maximum length" field allows you to limit the maximum length of a section, if possible. Once this value is exceeded, the aggregation process stops at the section concerned.

Script Command 
<associationwithfield>
    <int>Field name</int>
    <bool>1 to sort by subtype, otherwise 0</bool>
    <float>Minimum section length</float>
    <float>Maximum section length</float>
</associationwithfield>
