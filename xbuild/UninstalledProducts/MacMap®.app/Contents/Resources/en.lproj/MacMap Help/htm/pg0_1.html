MacMap® Help

The Application

MacMap® is a software application based on a core architecture to which external functions called "plug-ins" can be added. MacMap® also uses a dynamic library called "MacMapSuite" which is installed in the "Library/Frameworks/" folder of your system by the MacMap® installer. This library can be used by external software applications to access the contents of MacMap®v3 tables. The main types of external functions are "Tools", "XMaps", "Geogs", "Calcs" and "vDefs".
Tools

These are tools that allow you to manipulate objects and navigate in the cartographic window. You can access them from the "Toolbox".
XMaps

These are generic functions with no specific use (e.g. Select Objects, Spreadsheet...).
Geogs

These are the geographic search modules which can be integrated into a request (e.g. select the objects which are within a certain distance from a particular object, find the points included in a polygon, ...).
Calcs

These are calculation functions which return a result for a given object (e.g. length of a line, object ID, ...). Calcs can be compiled or defined by the user with the "Formula Editor".
Constants

These are calculation functions which return a constant result, for example the value of Pi or of e. Constants can be compiled or defined by the user with the "Constant Editor".
vDefs

These modules let you define a cartographic representation (or "style") for objects: an area can be displayed using a colored polygon, using the line defining its outline, using a point (e.g. barycenter) or a collection of points, using text (e.g. the name of this polygon)...
The vDefs are accessible from the "Presentation Box".
The styles are grouped into collections called "Views" which define a cartographic representation of the database.
Each type can be displayed as many times as necessary using different styles in the same view.
Windows

The main window: It displays the cartographic representation of the database.

The mini card: Located below the menu bar, it displays information about the selection.
