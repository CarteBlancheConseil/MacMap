XMap - Macros

The Macro box displays the list of macros saved with the document.
Clicking on one of the buttons lets you run the corresponding macro.
Clicking next to one of the buttons lets you select the macro.
Click the "+" button to add a macro.
Click the "-" button to delete the selected macro.
Click the "E" button to edit the selected macro.
Click the "D" button to duplicate the selected macro.
The creation/modification of macros is done via the "Select" function.
