Shear Tool
Use

The Shear tool lets you edit the segmentation of an object (line or area).
For lines, clicking lets you split an object into 2 parts each having the same attributes.
For area entities, a succession of clicks lets you split an object into 2 parts each having the same attributes. If all clicks are done on the area, you end up with an area with a hole, and a solid object corresponding to the hole. In this case, the operation is finished with a double-click or by pressing "Enter". If you start by clicking on the area border, the operation ends when you click a second time on the border or outside of the object.

This tool uses join options to make editing easier.
Popup

This tool does not have any popups.

Options

The options involve joins. You can enable or disable each of 4 types of joins and you can specify the types to be considered. The different joining procedures can be enabled by specifying an activation key in the fields next to the check boxes.
