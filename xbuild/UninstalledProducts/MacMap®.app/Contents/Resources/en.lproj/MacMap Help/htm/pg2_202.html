Retouch Tool
Use

The Retouch tool lets you edit the segmentation for an object (line or area).
First, click to select the object to be edited. Then, you can move each point in the segmentation individually.
To delete a point, hold down the "Alt" key and click on the point.
To add a point, hold down the "Apple" key and click on the segmentation.
This tool uses join options to ensure correct retouching.
Popup

This tool does not have any popups.

Options

The options involve joins. The "Join in the Direction" is not used. You can enable or disable each of 3 other types of joins and you can specify the types to be taken into account. The different joining procedures can be enabled by specifying an activation key in the fields next to the check boxes.
