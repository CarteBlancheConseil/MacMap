XMapBox - Draw II

The Draw II XMapBox includes the functions of Draw XMapBox in a simplified interface.
Joining functions are accessed directly using the first 4 buttons and the distance is set using a text field placed next to the button concerned when the box is fully open. For more information on joining, see the functions Join Options, Join At Shortest, Join At Angularities, Join At Ends, and Join In The Direction
The fifth button allows you to launch clean-up.
The next 2 buttons allow you to enable or disable the automatic cleanup and topological consistency functions, while the buttons placed next to them allow you to access the configuration of these functions.
The last button allows you to close/open the box.
