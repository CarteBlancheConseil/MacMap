Geog - Outside

Search for objects of the specified type located outside contrasted areas (or pre-selected if none exist).
The "Partially" option lets you retain all objects for which at least one point is located outside the areas.
The "Entirely" option lets you retain only those objects for which all points are located outside the areas.
The "Strictly" option lets you not retain objects for which some points are located on an area boundary.

Script Command 
<outside>
    <bool>Entirely (0/1)</bool>
    <bool>Strictly (0/1)</bool>
</outside>
