vDef - XML Editor

The XML Editor vDef lets you edit any style whether or not it has its own editor.
The use of this function is reserved for specialists. It requires that you ensure that the XML information entered is consistent and that you are familiar with the structure of MacMap速 styles.
When editing a style which has its own editor, it is better to edit the vDef signature (tag sign) so that you avoid later editing this style with a standard editor, thereby losing additional information that was entered.

Apply

This button saves the changes to the style and apply it to the current view if this style is shown at screen.
