XMap - Tool Menu Setup

This plug-in allows you to set up the tool context menu (right-click on the map), allowing access to tools depending on the operations you regularly use.
The list on the left presents the list of tools in the order they were loaded.
The buttons in the middle allow you to set the configuration (add, remove, order of appearance...).
The list on the right shows the current configuration.
