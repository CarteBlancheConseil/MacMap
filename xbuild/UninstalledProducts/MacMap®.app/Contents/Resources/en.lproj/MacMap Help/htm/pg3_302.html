XMap - Clean

The "cleanup" function lets you make automatic corrections to a network in order to make it topological. It uses Cleanup Options to perform 6 processes:

Create Nodes:
This process creates crossroads at intersections and ends of edges, unless the intersection is located in the action radius of a bridge.

Split Edges:
Edges are split at each intersection marked by a crossroads.

Join Edges on Nodes:
Ends of edges are joined to the nearest crossroads. This function lets you create a topological network.

Join Nodes on Edges:
Nodes are joined at the ends of the closest edges.

Examine Nodes:
This function lets you examine each crossroads and fill in the "Control" field with an 8-character code:
xxxxxxxx: nothing to indicate
1xxxxxxx: node created by splitting
x1xxxxxx: node joined to an edge by join
xx1xxxxx: isolated node
xxx1xxxx: redundant node (superimposed on another)
xxxxx1xx: node behind a dead-end
xxxxxx1x: node not joining 3 edges

Examine Edges:
This function lets you examine each edge and fill in the "Control" field with an 8-character code:
xxxxxxxx: nothing to indicate
1xxxxxxx: edge created by splitting
x1xxxxxx: edge joined to a node by join
xx1xxxxx: loop edge (starting point matches end point)
xxx1xxxx: dead-end edge
xxxx1xxx: edge completely isolated
xxxxx1xx: edge not joined to a node
xxxxxxx1: incorrect segmentation (reduced to a point).

The field named "Control" is automatically created if necessary. If you would like to create it yourself, it must be of type text and include a minimum of 8 characters.
