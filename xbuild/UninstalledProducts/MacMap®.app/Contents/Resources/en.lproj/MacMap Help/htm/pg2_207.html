Redraw Tool
Use

The redraw tool revectorizes a portion of the segmentation of an object (line or surface.
A first click allows you to select the object to be modified and to set the first cutting point. The second click allows you to set the second cutting point. You can then proceed to redraw the segmentation portion.
To delete the last point entered, press the delete key.
To cancel the editing in progress, press the esc key.
This tool uses join options.
Popup

This tool does not have any popups.

Options

The options involve joins. You can enable or disable each of 4 types of joins and you can specify the types to be considered. The different joining procedures can be enabled by specifying an activation key in the fields next to the check boxes.
