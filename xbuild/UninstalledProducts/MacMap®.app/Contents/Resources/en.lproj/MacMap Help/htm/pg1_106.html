XMap - ShapeFile Export

This function lets you export objects in ShapeFile (ArcView™) format.
If a selection exists, this selection is exported in as many tables as there are types. If there is no selection, all of the objects in the project are exported in as many tables as there are types. The ShapeFile tables generated have the same names as the types.
If the projection information (see Export format and Preferences) has been set, the data is dynamically reprojected on export.

Script Command 
<shpexport>
    <path>Folder path (POSIX)</path>
