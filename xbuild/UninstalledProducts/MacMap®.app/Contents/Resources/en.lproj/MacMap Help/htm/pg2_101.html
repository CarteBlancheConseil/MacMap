Selection Tool
Use

This tool lets you select document objects by using the mouse, with a single click or by area (rectangle, circle or polygon). For selections by polygon, the angle constraint can be activated using the "Shift" key.
If at the end of a selection, the "Shift" key is held down, the selection is added to the existing selection. Objects belonging to the initial selection and to the new selection are deselected.
If the "Shift" key is not held down, the selection is replaced.
Double-click to open the card on the selection.
Objects represented by unselectable styles cannot be selected (see Presentation Box).

Popup

The popup lets you choose the type of selection by area:
Selection by Rectangle: click and drag.
Selection by Circle: click and drag.
Selection by Polygon: a succession of clicks, then "Enter" or double-click to validate the selection.
If you chose the selection by polygon, selection by pointing is disabled. To enable selection by pointing, choose one of the other modes.

Options

4 options are available: Select All Objects: if several objects of the same type are superimposed, only the first object is selected if this option is disabled. Otherwise, all objects are selected.
Select All Types: if several objects of different types are superimposed, only the objects of the first type (the one most recently displayed in the presentation box) are selected if this option is disabled. Otherwise, all objects are selected.
Objects Completely Included: when selecting by zone, do not include objects whose segmentation points are included in the selected area.
Use Angle Constraint: when selecting by polygon, it lets you enable angle constraint for the desired value (expressed in degrees).

The two lists presented in the "Selection By Pointing" and "Selection By Area" sections let you filter the types that can be selected in these two modes. These filters are added to the pre-existing filtering in the presentation box.
