vDef - Histograms

This vDef permet allows you to represent data as histograms.
The pop-up, the list and the "+" and "-" buttons in the top, right-hand corner of the window, allow you to select fields to be included in the view.
Bars appearence

The elements of this topic lets you determine the color and transparency of bars for each series.

Dimensions

The elements of this section are used to define the general appearance of the histogram: Height and width of the area, bar spacing and orientation, x and y offset.

Background

The elements of this topic lets you determine the appearance of the background, if the "Background" check box is on.

Statistics

The "Calculate min/max" popup lets you calculate min and max automatically (based on type, selection or contrasts).
The "own scale" checkbox is used to determine whether each data must be represented with its own min / max (own scale) or the min / max of all series combined (common scales).
The two text boxes allow you to directly modify the min and max desired.

Visibility

The two popups let you set the scale range for which the style is visible.

Options: General

The "Scale of Reference" field lets you set the proportionality based on the scale. Each of the sizes indicated is recalculated when displayed according to the current scale and reference scale.
The "Unit" field lets you select between various measurements (size, thickness, ...) in pixels or in millimeters.
The "CMYK" field lets you set the type of color you want: CMYK or RGB.

Options: Display object if

This option lets you define display conditions for each object. A display condition is made up of three elements:
- A field in the attribute table,
- An operator,
- A value.
An object not meeting the defined condition will not be displayed by the style, without having to hide it.

Options: Offset

This option lets you select two x and y fields which modify the location of the display without modifying the object geometry. These fields can be filled in using the Offset tool.

Apply

This button saves the changes to the style and apply it to the current view if this style is shown at screen.
