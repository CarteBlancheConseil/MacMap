vDef - Text View

The Text View vDef lets you display the contents of a field in text format. By default, the style is based on the modes of the "Subtype" field.

General Settings

The options in the "General Settings" section are common to all modes of the displayed field.
The "Display Field" popup lets you select the field to display. The "Display at Centroid" option lets you display only a single point at the centroid of the object represented. If this option is not checked, line and area objects will be represented by text that follows the segmentation.

Font

The "Font" button lets you open the "Font" palette and select the character font to be used.
The "Size" field lets you define the size of the text.
The "Text Color" field lets you specify the color of the text.
The "Thickness" field lets you define the size of the clipping.
The "Outline Color" field lets you specify the color of the clipping.

Background

The "Opacity" field lets you adjust the transparency of the background.
The "Interior" field lets you specify the color of the background.
The "Thickness" field lets you set the thickness of the background contour.
The "Vertical Margin" field lets you set a vertical margin between the text and the edge of the background.
The "Horizontal Margin" field lets you set a horizontal margin between the text and the edge of the background.
The "Rounding Radius" field lets you specify a radius for obtaining a background in the shape of a rounded rectangle.
The "Line Offset" field lets you offset the background.

Position

The "Alignment" pop-up lets you select the text alignment (left, centered, right or justified).
The "Letter Spacing" field lets you set the spacing between letters.
The "Word Spacing" field lets you set the spacing between words.
The "Rotation" field lets you specify an angle for displaying text at the centroid.
The "Line Offset" field lets you offset the text.
The "dx" field lets you set a horizontal offset with respect to the text anchoring point.
The "dy" field lets you set a vertical offset with respect to the text anchoring point.
The "Smoothing" field allows you to remove angularities when displaying text along a segmentation.
The "Readable" field allows you to orientate displayed text so that the user can always read it.

Visibility

The settings of the "Visibility" section apply individually, for each mode of the displayed field.
The two popups let you set the scale range for which the style is visible.

Options: General

The "Class Field" field lets you specify the information that the style must represent. Only numerical fields which have entry constraints can be used.
If you change the class field of an existing style, the display settings of each of the modes are reset.
The "Scale of Reference" field lets you obtain a proportional view. Each of the sizes indicated will be recalculated when displayed according to the current scale and reference scale. Setting this field to a value of "Fixed Size" results in an unchanged display regardless of the current scale.
The "Unit" field lets you select between various measurements (size, thickness, ...) in pixels or in millimeters.
The "Reverse Display" field lets you reverse the order in which classes are displayed.
The "CMYK" field lets you set the type of color you want: CMYK or RGB.

Options: Display object if

This option lets you define display conditions for each object. A display condition is made up of three elements:
- A field in the attribute table,
- An operator,
- A value.
An object not meeting the defined condition will not be displayed by the style, without having to hide it.

Options: Offset

This option lets you select two x and y fields which modify the location of the display without modifying the object geometry. These fields can be filled in using the Offset tool.

Apply

This button saves the changes to the style and apply it to the current view if this style is shown at screen.
