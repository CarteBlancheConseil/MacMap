XMap - Import Raster

Select the image to import (TIF, JPEG, PNG or GIF).
If this image uses valid georeferencing settings (TAB or TFW formats), the import is done automatically.
If the image does not have any georeferencing settings, the function will ask you to define the upper left corner and lower right corner coordinates or the scale of the image.
The objects are imported as rectangles corresponding to the image area, in the Raster type defined in the import grid. If you resize the base rectangle, the image will be resized accordingly. The image name is defined in the Object Name field.
The images are duplicated in the corresponding table (Contents/Resources/rasters/ folder).
To remain geographically consistent, an imported image must be in the same projection as the database you are working with.
