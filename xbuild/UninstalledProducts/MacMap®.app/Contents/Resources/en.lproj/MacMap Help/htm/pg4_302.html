XMap - Create Polygon

This function lets you create a polygon in the type selected in the presentation box starting from a centroid, a radius and a number of sides. If an object is contrasted, the centroid is initialized to the coordinate of the first point of that object; otherwise, it is initialized to the coordinate at the center of the screen.

Script Command 
<makepoly>
    <name>Type name</name>
    <float>X</float>
    <float>Y</float>
    <float>Size</float>
    <int>Number of Sides</int>
</makepoly>
