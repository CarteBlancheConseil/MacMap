XMap - Type Manager

The type manager allows all operations on the types in a database. The list shows the types present in the database.
The buttons allow the following operations:
- New: creates a type. This is a shortcut to the Create Type function.
- Open: opens an existing type. This is a shortcut to the Open Type function.
- Remove: indicates that the selected type must be removed when the database is closed
- Rename: indicates that the selected type must be renamed when the database is closed
- Protect: prevents any modifications to the selected type.
- Unprotect: authorizes modifications to the selected type.
