XMap Gestionnaire de tirets

Le gestionnaire de tirets permet de d'importer des fichiers XML décrivant des tirets dans un type. Ces tirets peuvent ensuite être utilisés dans les vDef.
Le bouton "+" permet d'ouvrir un fichier xml stocké sur disque.
La liste permet de choisir un tiret. Sa représentation est affichée dans la zone inférieure de la fenêtre.
En cliquant sur choisir, on retourne le tiret à la vDef appelante.

Liste des vDefs utilisant des tirets :
Vue lignes
Lignes proportionnelles
Lignes colorés
