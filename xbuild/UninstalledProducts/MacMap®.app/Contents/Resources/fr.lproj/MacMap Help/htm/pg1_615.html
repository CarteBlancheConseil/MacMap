XMap Lien Multimédia

Le lien multimédia permet d'associer une URL à chaque objet, et d'en afficher le contenu dans une fenêtre flottante. L'adresse doit être indiquée dans un champ nommé "URL", et elle peut désigner une ressource locale sur disque, ou sur un réseau internet/intranet.
Pour une ressource locale (sur disque), l'URL sera du type : file:///users/mon_nom/Séquences/mon_film.avi, par exemple, si l'on veut associer une vidéo à un objet.
Pour une ressource internet, l'URL sera du type : http://www.monsite.com, par exemple, si l'on veut associer un site web à un objet.
Les URL peuvent désigner des images, des sites web, des séquences vidéos, des séquences audio, ou encore du texte brut.
