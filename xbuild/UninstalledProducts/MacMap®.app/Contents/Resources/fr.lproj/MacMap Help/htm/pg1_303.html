XMap Transfert de type

Les options permettent d'effacer la sélection après transfert, et de créer une structure de champs similaire dans le type récepteur.
Cette fonction opère sur une sélection monotype.

Commande script 
<transtype>
    <name>Nom du type</name>
    <bool>1 pour conserver les objets originaux, 0 pour les supprimer</bool>
    <bool>1 pour créer les champs, 0 sinon</bool>
</transtype>
