vDef Vue textes

La vDef Vue textes permet d'afficher le contenu d'un champ sous forme de texte. Par défaut, le style est basé sur les modalités du champ "Sous-type".

Paramètres généraux

Les options de la rubrique "Paramètres généraux" sont communes à toutes les modalités du champ affiché.
Le pop-up "Afficher le champ" permet de choisir le champ à afficher. L'option "Affichage au centroïde" permet de n'afficher qu'un seul point au centroïde de l'objet représenté. Si cette option n'est pas cochée, les objets linéaires et surfaciques seront représenté par un texte épousant la segmentation.

Caractères

Le bouton "Police" permet d'ouvrir la palette "police" et de choisir la police de caractère à utiliser.
Le champ "Taille" permet de définir la taille du texte.
Le champ "Couleur texte" permet de spécifier la couleur du texte.
Le champ "Epaisseur" permet de définir la taille du détourage.
Le champ "Couleur contour" permet de spécifier la couleur du détourage.

Fond

Le champ "Opacité" permet de régler la transparence du fond.
Le champ "Intérieur" permet de choisir la couleur du fond.
Le champ "Epaisseur" permet de définir l'épaisseur du contour du fond.
Le champ "Marge horizontale" permet de définir marge entre le texte et le bord du fond, horizontalement.
Le champ "Marge verticale" permet de définir marge entre le texte et le bord du fond, verticalament.
Le champ "Rayon d'arrondi" permet de spécifier un rayon pour obtenir un fond de forme rectangulaire arrondie.
Le champ "Décalage de ligne" permet d'obtenir un décalage du fond.

Positionnement

Le pop-up "Alignement" permet de choisir l'alignement du texte (à gauche, centré, à droite, ou justifié).
Le champ "Inter-lettres" permet de régler l'espacement entre les lettres.
Le champ "Inter-mots" permet de régler l'espacement entre les mots.
Le champ "Rotation" permet d'indiquer un angle pour l'affichage des textes au centroïde.
Le champ "Décalage de ligne" permet d'obtenir un décalage du texte.
Le champ "dx" permet définir un offset horizontal par rapport au point d'accroche du texte.
Le champ "dy" permet définir un offset vertical par rapport au point d'accroche du texte.
Le champ "Lissage" permet d'éliminer les angularités dans l'affichage des textes le long d'une segmentation.
Le champ "Lisible" permet de d'orienter l'affichage du texte de façon à ce qu'il soit toujours lisible par l'utilisateur.

Visibilité

Les paramètres de la rubrique "Visibilité" s'appliquent individuellement, pour chaque modalité du champ affiché.
Les deux pop-up permettent de régler la plage d'échelles pour lesquelles le style est visible.

Options : Généralités

Le champ "Champ classe" permet de spécifier l'information que le style doit représenter. On ne peut utiliser que des champs numériques disposant de contraintes de saisies.
Si l'on change le champ classe d'un style pré-éxistant, les paramétres d'affichages de chacune des modalités sont réinitialisés.
Le champ "Echelle de référence" permet d'obtenir une vue proportionelle. Chacune des tailles indiquées sera re-calculée lors de l'affichage en fonction de l'échelle en cours et de l'échelle de référence. En laissant ce champ à la valeur "Taille fixe", on obtient un affichage inchangé quelque soit l'échelle en cours.
Le champ "Unité" permet de choisir entre une expression des mesures (tailles, épaisseurs, ...) en pixels ou en millimètres.
Le champ "Affichage inverse" permet d'inverser l'ordre d'affichage des classes.
Le champ "CMYK" permet définir le type de couleur désiré : CMYK ou RGB.

Options : Afficher l'objet si

Cette option permet de fixer des conditions d'affichage, pour chaque objet. Une condition d'affichage se compose de trois éléments :
- Un champ de la table attributaire,
- Un opérateur,
- Une valeur.
Un objet ne répondant pas à la condition définie ne sera pas affiché par le style, sans avoir à le masquer.

Options : Offset

Cette option permet de choisir deux champs x et y qui modifient l'emplacement de l'affichage sans pour autant modifier la géométrie de l'objet. Ces champs peuvent être renseignés en utilisant l'outil Offset.

Appliquer

Ce bouton permet d'enregistrer les modifications apportées au style et de l'appliquer à la vue en cours si ce style est représenté à l'écran.
