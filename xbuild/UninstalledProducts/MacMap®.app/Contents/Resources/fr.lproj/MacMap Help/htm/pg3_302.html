XMap Nettoyer

Le nettoyage permet de réaliser des corrections automatiques sur un réseau, dans le but de le rendre topologique. Il utilise les Options de nettoyage pour réaliser 6 traitements :

Créer les noeuds :
Ce traitement créé les carrefours aux intersections et extrémités des arêtes, sauf si l'intersection se trouve dans le rayon d'action d'un pont.

Découper les arêtes :
Les arêtes sont découpées à chaque intersection matérialisée par un carrefour.

Joindre les arêtes sur les noeuds :
Les extrémités des arêtes sont jointes sur le carrefour le plus proche. Cette fonction permet de rendre un réseau topoloqique.

Joindre les noeuds sur les arêtes :
Les noeuds sont joints sur les extrémités d'arêtes les plus proche.

Examiner les noeuds :
Cette fonction examine chaque carrefour et renseigne le champ "Contrôle" avec un code à 8 caractères :
xxxxxxxx : rien à signaler
1xxxxxxx : noeud créé par découpage
x1xxxxxx : noeud joint à une arête par jonction
xx1xxxxx : noeud isolé
xxx1xxxx : noeud redondant (superposé à un autre)
xxxxx1xx : noeud en fond d'impasse
xxxxxx1x : noeud ne joignant pas 3 arêtes

Examiner les arêtes :
Cette fonction examine chaque arête et renseigne le champ "Contrôle" avec un code à 8 caractères :
xxxxxxxx : rien à signaler
1xxxxxxx : arête scindée par découpage
x1xxxxxx : arête joint à un noeud par jonction
xx1xxxxx : arête bouclée (tenant égal à l'aboutissant)
xxx1xxxx : arête en impasse
xxxx1xxx : arête entièrement isolée
xxxxx1xx : arête non jointe à un noeud
xxxxxxx1 : segmentation incorrecte (réduite à un point).

Le champ nommé "Contrôle" est créé automatiquement si nécessaire. Si vous désirez le créer vous-même, il doit être de type texte et comporter 8 caractères au minimum.
