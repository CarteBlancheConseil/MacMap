vDef Raster

Permette di visualizzare delle immagini raster, regolando la visualizzazione e l'opacità per ciascun sotto-tipo.
Grafica

La prima parte della finestra permettde di definire alcuni parametri di visualizzazione :
- Mostra : per visualizzare o no i rastre di un sotto-tipo.
- Opacité : regola la trasparenza del raster di ciascun sotto-tipo.
Vista

I parametri della rubrica "Vista" si applicano individualmente, per ciascun Sotto-tipo visualizzato.
I due pop-up permettono di regolare la sequenza delle scale alle quali lo stile sarà visibile.

Opzioni : Parametri generali

Il campo "Campo classe" permette di specificare l'informazione che lo stile deve rappresentare. Si possono utilizzare solo dei campi numerici che dispongono di valori predefiniti.
Se si cambia il campo classe di uno stile già esistente, i parametri di visualizzazione vengono reinizializzati.
Il campo "Inverti l'ordine" permette di invertire l'ordine di visualizzazione delle classi.

Opzioni : Mostra l'oggetto se

Questa opzione permette di fissare le condizioni di visualizzazione, per ciascun oggetto. Una condizione di visualizzazione si compone di tre elementi :
- Un campo del formato scheda,
- Un operatore,
- Un valore.
Se l'oggetto non risponde alle condizioni definite non sarà visualizzato dallo stile, senza necessità di essere mascherato.

Applica

Registra le modifiche allo stile.
Se lo stile è rappresentato nella vista in corso, le modifiche sono subito applicate, senza la necessità di uscire dall'editor.
Per uscire dall'editor senza salvare le modifiche è sufficiente chiuderlo o premere "esc" sulla tastiera.
