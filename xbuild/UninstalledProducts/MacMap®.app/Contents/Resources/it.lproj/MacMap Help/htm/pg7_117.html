vDef Client WMS

La vDef Client WMS permette di visualizzare dei raster per richiesta tramite un server WMS, sulla superficie di disegno (schermo o area di stampa).
Servizio

La prima parte della finestra permette di definire l'accesso al server WMS :
- URL : indica l'url su cui si trova il server WMS.
- Connessione : permette di stabilire la connessione con il server (richiesta GetCapabilities).
- Esporta : permette di stabilire la connessione con il server e di recuperare l' XML di GetCapabilities in un documento, per l'analisi manuale.

Layers

Questa lista contiene i layer descritti nel GetCapabilities. Selezionate un layer per visualizzarlo con questo stile.

Grafica

I due pop-up permettono di impostare le scale su cui lo stile è visibile.

Applica

Registra le modifiche allo stile.
Se lo stile è rappresentato nella vista in corso, le modifiche sono subito applicate, senza la necessità di uscire dall'editor.
Per uscire dall'editor senza salvare le modifiche è sufficiente chiuderlo o premere "esc" sulla tastiera.
