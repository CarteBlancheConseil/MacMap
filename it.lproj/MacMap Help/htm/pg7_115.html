vDef Editor XML

La vDef Editor XML permette di definire qualsiasi stile, anche se non dispone di un editor proprio.
L'utilizzo di questa funzione è riservata agli utenti esperti. Per il suo corretto funzionamento, i parametri XML inseriti devono essere coerenti e devono rispettare la struttura degli stili MacMap®.
Se lo stile dispone di un proprio editor, è meglio cambiare il nome della vDef (tag sign), per evitare di definire in un secondo tempo questo stile con un editor standard, perdendo in questo modo le informazioni supplementari inserite.

Applica

Registra le modifiche allo stile.
Se lo stile è rappresentato nella vista in corso, le modifiche sono subito applicate, senza la necessità di uscire dall'editor.
Per uscire dall'editor senza salvare le modifiche è sufficiente chiuderlo o premere "esc" sulla tastiera.
