XMap - Shortcuts

This utility lets you create a shortcut to a MacMap速 function.
The first pop-up lets you select the kind of function: XMap, XMapBox, Script or Macro.
The functions of the corresponding type are displayed in the following list.
The special keyboard shortcuts are obtained using the following 4 check boxes (Shift, Ctrl, Alt and Cmd).
Finally, the character is indicated in the text field. The character code is indicated in the static text alongside this field.
The Apply button lets you assign the specified shortcut to the function selected from the list.
