XMap - Join Data

Allows you to join external data contained in files in CSV, tabulated text or DBF formats. The fields contained in the external table are added to the series of attribute data. These fields cannot be edited. However, they can be deleted from the list of attribute fields (without modifying the file).
To make a join, all you need to do is define a key field between the MacMap速 table concerned and the external table.
