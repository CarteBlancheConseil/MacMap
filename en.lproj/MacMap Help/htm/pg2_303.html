Rotate Symbol Tool
Use

The Rotate Symbol tool lets you use the mouse to individually adjust the display angle of the symbols and text of the objects represented by the Symbol View and Text View vDefs. To do this, you must first specify a field for the "Rotation" section.
A click and drag lets you modify the value of the object field and its representation.

Popup

This tool does not have any popups.

Options

This tool lets you use an angle constraint.
