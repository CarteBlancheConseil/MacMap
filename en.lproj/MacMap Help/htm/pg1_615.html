XMap - Multimedia Link

The multimedia link lets you associate a URL with each object and display the contents in a floating window. The address must be indicated in a field named "URL" and it can designate a local resource on disk or on an internet/intranet network.
For a local resource (on disk), the URL will be of type: file:///users/my_name/Sequences/my_movie.avi, for example, if you want to associate a video with an object.
For an internet resource, the URL will be of type: http://www.mysite.com, for example, if you want to associate a web site with an object.
URLs can designate images, websites, video sequences, audio sequences or even raw text.
