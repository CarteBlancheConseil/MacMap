XMap - Export Format

This allows you to determine the list of attributes of the database to be exported, by type.
The fields available are presented on the list on the left, the fields exported on the list on the right.
The buttons in the central area make it possible to alter the list of exported fields.
MacMap速 Text, ShapeFile, Mif-Mid and KML exports currently use this option.
The "Projection" button allows you to access a window where you can set the outgoing data projection. If the document projection has been set, this allows you to dynamically reproject the data on export.
Script Command

    <int>EPSG projection identifier</int>
    <!--List of type definitions-->
    <container>
        <name>Type name</name>
        <!--List of fields for export-->
        <string>Field name</string>
        <!--This tag must be repeated for each field to export-->
    </container>
    <!--This sequence must be repeated for each type definition-->
</fexp>
