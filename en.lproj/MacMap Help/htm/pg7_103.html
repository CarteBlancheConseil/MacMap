vDef - Line View

The Line View vDef lets you represent line or area objects with a line or a series of lines. By default, the style is based on the modes of the "Subtype" field.

Appearance

The settings of the "Appearance" section apply individually, for each mode of the displayed field.
You can define up to three passages (exterior, central and interior) for the same mode.
The "Thickness" field lets you specify the line thickness.
The "Cap" field lets you select the type of line ending (butt, round or projecting).
The "Join" field lets you select the representation type for each of the joins of the segmentation (miter, rounded or bevel).
The "Dash" field lets you select a dash style.
The "Color" field lets you specify the color of the line.
The "Miter Limit" field lets you set the limit at which joins are beveled.
The "Offset" field lets you offset the line (to the right for a positive value and to the left for a negative value).
The "Smoothing" field lets you smooth the segmentation (remove angularities).

Visibility

The settings of the "Visibility" section apply individually, for each mode of the displayed field.
The two popups let you set the scale range for which the style is visible.

Options: General

The "Class Field" field lets you specify the information that the style must represent. Only numerical fields which have entry constraints can be used.
If you change the class field of an existing style, the display settings of each of the modes are reset.
The "Scale of Reference" field lets you obtain a proportional view. Each of the sizes indicated will be recalculated when displayed according to the current scale and reference scale. Setting this field to a value of "Fixed Size" results in an unchanged display regardless of the current scale.
The "Unit" field lets you select between various measurements (size, thickness, ...) in pixels or in millimeters.
The "Reverse Display" field lets you reverse the order in which classes are displayed.
The "CMYK" field lets you set the type of color you want: CMYK or RGB.
Options: Display object if

This option lets you define display conditions for each object. A display condition is made up of three elements:
- A field in the attribute table,
- An operator,
- A value.
An object not meeting the defined condition will not be displayed by the style, without having to hide it.

Options: Offset

This option lets you select two x and y fields which modify the location of the display without modifying the object geometry. These fields can be filled in using the Offset tool.

Apply

This button saves the changes to the style and apply it to the current view if this style is shown at screen.
