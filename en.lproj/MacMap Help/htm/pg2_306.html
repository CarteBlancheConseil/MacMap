Line Offset Tool
Use

The Line Offset tool lets you individually offset the display of lines represented by the Line View vDef, text represented by the Text View vDef and arrows represented by the Orientation View vDef. To do this, you must first specify a field for the "Offset" section. This tool can not be used to modify the segmentation of objects.

Popup

This tool does not have any popups.

Options

This tool has no options.
