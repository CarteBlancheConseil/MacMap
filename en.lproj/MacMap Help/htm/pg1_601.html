XMap - Card

The card lets you view and modify the attribute table for one or more selected objects. Each field can be individually modified. The card can be accessed from a menu, or by double-clicking with certain tools (Select, Tooltip).

By default, fields are editable in text mode. In this mode, the "Enter" key lets you validate the modification in progress; the "Esc" key is used to end the edit without saving it.
Fields with entry constraints are editable via a popup.
Boolean fields are editable with a checkbox.
Hidden fields are not visible.
Protected fields are not editable.
Fields can be organized by clicking on the field heading while holding down the Option ("Alt") key.

The "<", "<<", ">" and ">>" buttons let you scroll through the selected objects. The "Selected" checkbox lets you remove or add an object to the selection. The "Reorganize" button lets you reposition the fields in their initial order.

The "Esc" key lets you close the card if no editing is in process. The right arrow lets you move to the next object. The left arrow lets you move to the previous object. The Alt-Right Arrow keyboard shortcut lets you move to the last object. The Alt-Left Arrow keyboard shortcut lets you move to the first object. The Alt-Return keyboard shortcut lets you select/deselect the current object. The tab key lets you move to the next field.
