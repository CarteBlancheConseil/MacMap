vDef - WMS Client

The WMS Client vDef allows you to display rasters on the drawing area (screen or printing area) through requests to WMS servers.
Service

The first part of the window allows you to set access to the WMS server:
- URL: indication of the url where the WMS server is located.
- Connection: allows you to establish a connection with the server (GetCapabilities request).
- Export: allows you to establish a connection with the server and retrieve the GetCapabilities XML in a file, for manual analysis.
Layers

This list contains the layers described in GetCapabilities. Select a layer to display with that style.

Visibility

The two popups let you set the scale range for which the style is visible.

Apply

This button saves the changes to the style and apply it to the current view if this style is shown at screen.
