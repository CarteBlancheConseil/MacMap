XMap - Create Type

This function lets you create a new type at the desired location.
The options let you select the kind (Point, Line, Area, Text, or Raster) as well as the type of storage (currently in MacMap速v3 format or in a PostGIS database).
The type inherits the resolution, unit of measure and projection of the current project.
The "Add to view" option lets you add a style directly to the current view.
