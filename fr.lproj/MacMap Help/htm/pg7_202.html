XMap Gestionnaire de motifs

Le gestionnaire de motifs permet de d'importer des fichiers pdf dans un type. Ces motifs peuvent ensuite être utilisés dans les vDef pour générer des trames.
Le bouton "+" permet d'ouvrir un fichier pdf stocké sur disque.
La liste permet de choisir un motif. Sa représentation est affichée dans la zone inférieure de la fenêtre.
En cliquant sur choisir, on retourne le motif sélectionné à la vDef appelante.

Liste des vDefs utilisant des motifs :
Vue points
Vue rectangles
Vue orientation
Vue surfaces
Points proportionnels
Rectangles proportionnels
Points colorés
Rectangles colorés
Surfaces choroplèthes
Editeur de gradient
