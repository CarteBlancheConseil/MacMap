Geog Le long

Recherche les objets du type linéaire spécifié se trouvant le long les objets contrastées (ou à défaut pré-sélectionnées).
L'option "en partie" permet de retenir tous les objets dont au moins deux points consécutifs se trouvent le long des objets sources.
L'option "en totalité" permet de ne retenir que les objets dont tous les points se trouvent le long des objets sources.
L'option "strictement" permet de ne pas retenir les objets dont certains points ne sont pas précisément sur la limite d'un des objets source.

Commande script 
<along>
    <bool>En totalité (0/1)</bool>
    <bool>Strictement (0/1)</bool>
</along>
