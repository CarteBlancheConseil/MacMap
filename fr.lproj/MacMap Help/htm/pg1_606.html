XMap Macros

La boîte macros affiche la liste des macros enregistrées dans le document.
Un clic sur l'un des boutons permet d'exécuter la macro correspondante.
Un clic en face d'un des boutons permet de sélectionner la macro.
Le bouton "+" permet d'ajouter une macro.
Le bouton "-" permet de supprimer la macro sélectionnée.
Le bouton "E" permet de modifier la macro sélectionnée.
Le bouton "D" permet de dupliquer la macro sélectionnée.
La création/modification des macros est assurée par la fonction "Sélectionner".
