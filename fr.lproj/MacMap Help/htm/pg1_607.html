XMap Calcs

La boîte Calcs affiche la liste des calcs disponibles.
Un clic sur l'un des boutons permet d'exécuter la calc correspondante. Une calc s'applique sur une sélection.
Un clic en face d'un des boutons permet de sélectionner la calc.
Le bouton "+" permet d'ajouter une calc utilisateur.
Le bouton "-" permet de supprimer la calc utilisateur sélectionnée.
Le bouton "E" permet de modifier la calc utilisateur sélectionnée.
Le bouton "D" permet de dupliquer la calc utilisateur sélectionnée.
Le bouton "∑" permet de chosir une fonction pour les sélections multiples : Somme, Moyenne, Médiane, Variance, Ecart type, Minimum ou Maximum.

La création/modification des calcs utilisateurs est assurée par l'"Editeur de formules".
