XMap Raccourcis clavier

Cet utilitaire vous permet d'affecter un raccourci clavier à une fonction MacMap®.
Le premier popup vous permet de sélectionner le genre de fonction : XMap, XMapBox, Script ou Macro.
Les fonctions du genre correspondant sont affichées dans la liste suivante.
Les combinaisons de touches spéciales sont obtenues à l'aide des 4 cases à cocher suivantes (Shift, Ctrl, Alt et Cmd).
Enfin, le caractère est indiqué dans la zone de texte. Le code caractère est indiqué dans le texte statique situé à coté de cette zone.
Le bouton "Appliquer" permet d'affecter le raccourci défini à la fonction sélectionnée dans la liste.
