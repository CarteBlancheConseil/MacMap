vDef Client WMS

La vDef Client WMS permet d'afficher des raster par requête auprès de serveurs WMS, sur la surface de dessin (écran ou zone d'impression).
Service

La première partie de la fenêtre permet de définir l'accès au serveur WMS :
- URL : indication de l'url à laquelle se trouve le serveur WMS.
- Connexion : permet d'établir la connexion avec le serveur (requête GetCapabilities).
- Exporter : permet d'établir la connexion avec le serveur et de récupérer le XML du GetCapabilities dans un fichier, pour analyse manuelle.
Layers

Cette liste contient les layers décrits dans le GetCapabilities. Sélectionnez un layer pour l'afficher avec ce style.

Visibilité

Les deux pop-up permettent de régler la plage d'échelles pour lesquelles le style est visible.

Appliquer

Ce bouton permet d'enregistrer les modifications apportées au style et de l'appliquer à la vue en cours si ce style est représenté à l'écran.
